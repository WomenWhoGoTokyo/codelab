# Women Who Go Tokyo コードラボについて

ここにあるコードラボは Women Who Go Tokyo の運営メンバーによって作られたものです。
主に Go に関する内容のものになります。

何か問題や間違いがありましたら、[issue](https://github.com/WomenWhoGoTokyo/codelab/issues) を起票してください。
